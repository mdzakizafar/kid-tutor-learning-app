# Kid Tutor – AI-Powered Adaptive Quiz Platform

## Overview

Kid Tutor is an interactive learning platform that generates
topic-based quizzes using the Gemini API and automatically adjusts
question difficulty according to user performance.

## Features

- AI-generated topic-based quizzes
- Easy, Medium, and Hard difficulty levels
- Adaptive difficulty progression
- Instant answer feedback
- AI-generated explanations
- Automated scoring
- Quiz progress tracking
- Leaderboard
- Responsive interface
- Functional contact form

## Tech Stack

- HTML
- CSS
- JavaScript
- Gemini API
- Vercel Serverless Functions
- Vercel
